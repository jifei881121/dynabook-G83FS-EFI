##Dynabook G83/FS-Hackintosh
### 适用：`i3，i5，i7版本`
## 电脑配置
| 电脑型号|       Dynabook G83/FS 笔记本电脑           |
| ------------- | ----------------------------------- |
|操作系统|  macOS-Sonoma  |
| CPU   |      英特尔 酷睿 i5 - 10210U / i7-10710U /i3-10110U
| 显卡| Intel UHD 620 DeviceID：9B41|
| 网卡 | IntelAX200/Intel-i219-V |
|内存 | DDR4 2666/3200笔记本内存 |
| 硬盘 | 默认三星256G硬盘可以更换其他m.2硬盘 |
| 显示器| 13 英寸 东芝TOS508F |
| 声卡 | Realtek ALC257 |
## 正常工作
- 显卡

- 电源

- 蓝牙

- 显示器亮度调节

- 无线（[原网卡AX200]/有线网卡

- 触摸板 

- USB定制 type-c可以使用

- 声卡
